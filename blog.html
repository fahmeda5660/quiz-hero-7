2. Difference between LocalStorage, Session Storage and Cookies?

Answer:

  • Local Storage stores data in browser without any expiration date, if the browser is closed than also data is not deleted, data can be used in future sessions. It's storage location: Browser only. Browers available in HTML5. Capacity: 10mb

  • Session Storage stores data in browser for a session (when tab is closed), where browser is closed, data is deleted, only available for one session. It's storage location: Browser only. Browers available in HTML5. Capacity: 5mb

  • Cookies expire date can set manually. It's storage location: Browser and server. Browers available in HTML4 and HTML5. Capacity: 4kb
